# CapCut Editing Playbook

## 1) Project setup
- Aspect ratio: 9:16
- Resolution: 1080x1920
- Frame rate: 30fps
- Safe margins: keep key text inside center 80% width

## 2) Timing recipe (30–45s short)
- 0:00–0:02: Hook text + face/frame movement
- 0:02–0:06: Problem line
- 0:06–0:14: Product/context proof
- 0:14–0:24: Step-by-step value
- 0:24–0:32: Safety/expectation line
- 0:32–0:40: CTA
- 0:40–0:45 (optional): end card + URL

## 3) Caption style
- Font: Inter Bold / Montserrat SemiBold
- Size: 54–68 (depends on shot density)
- Case: sentence case for readability; ALL CAPS only for 1-3 words
- Color: white text + 70% black shadow box
- Highlight color: #7BA4FF for keywords ("secure", "documented", "clear path")
- Position: lower-third center, avoid mouth/important UI
- Max words per line: 5-7

## 4) Typography system
- Hook card: 72-84, bold
- Body overlays: 54-64, semibold
- Legal/expectation line: 40-48, medium
- CTA line: 58-68, bold

## 5) Pacing and cuts
- Average shot length: 1.8-2.6s in first 10s
- Mid-section shot length: 2.5-3.5s
- Keep a visual change every 2-3s minimum
- Use punch-in (105-112%) for emphasis instead of flashy effects
- Transition style: hard cuts only; avoid gimmick transitions

## 6) Audio levels
- Voiceover/dialogue peak: -3 dB
- Voice average target: -9 to -6 dB
- Background music average: -24 to -20 dB
- Music ducking under speech: enabled
- Add light limiter on master to prevent clipping

## 7) Color and clarity
- Contrast: +8 to +12
- Saturation: +4 to +8
- Sharpen: +10 max
- Skin tone: keep natural (avoid heavy LUT look)
- UI clips: slight brightness lift (+4 to +6)

## 8) Export settings
- MP4 (H.264)
- 1080p, 30fps
- Bitrate: 12-16 Mbps
- Audio: AAC, 48kHz, 320kbps
- File naming: YYYY-MM-DD_topic_platform_v1

## 9) Thumbnail text rules
- 3-6 words max
- One promise only (example: "You Are Not Crazy")
- High contrast: white + blue accent + dark background
- Keep face + one key phrase visible
- Never claim guaranteed outcomes or legal promises

## 10) Compliance guardrails (always keep)
- Include one of these lines in either video text or caption:
  - "Educational support only."
  - "Not legal advice."
  - "No guaranteed outcomes."
- Avoid outcome promises (e.g., "guaranteed deletion", "instant score boost").

## 11) Fast production workflow (per clip)
1. Import master A-roll + UI clips.
2. Drop script markers by sentence.
3. Cut dead air and breaths >0.35s.
4. Add captions and keyword highlights.
5. Add B-roll/UI overlays to cover jump cuts.
6. Mix audio to target levels.
7. Add compliance line + CTA end card.
8. Export TikTok version, then duplicate for Shorts title/ending tweaks.
